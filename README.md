# Spotify Clone 🎵

A simple Spotify-like music player built using HTML, CSS, and JavaScript.

## Features
- Play / Pause songs
- Next & Previous controls
- Autoplay next song
- Progress bar
- UI sync between main and song list controls

## Tech Stack
- HTML
- CSS
- JavaScript

## Learning Outcome
- DOM Manipulation
- JavaScript Events
- Audio API
